# Iconos del Juego - Bolso de Emergencia

## Especificaciones de los Iconos

**Formato:** WebP
**Tamaño:** 128x128 píxeles
**Estilo:** Flat design, minimalista, colorido

## Lista de Iconos Necesarios (16 total)

### Categoría ESENCIAL (E) - 8 iconos
1. `water.webp` - Botella de agua
2. `food.webp` - Comida enlatada/paquetes
3. `flashlight.webp` - Linterna
4. `batteries.webp` - Pilas/baterías
5. `medkit.webp` - Botiquín de primeros auxilios (con cruz roja)
6. `whistle.webp` - Silbato de emergencia
7. `radio.webp` - Radio portátil
8. `documents.webp` - Documentos/identificación

### Categoría RECOMENDABLE (R) - 4 iconos
9. `mask.webp` - Mascarilla médica
10. `blanket.webp` - Manta/sleeping bag
11. `money.webp` - Billetes/efectivo
12. `keys.webp` - Llaves de casa

### Categoría NO ESENCIAL (N) - 4 iconos
13. `laptop.webp` - Laptop/computadora
14. `hairdryer.webp` - Secador de pelo
15. `book.webp` - Libro grueso
16. `plushie.webp` - Peluche/teddy bear

## Instrucciones para Generar

1. Crear sprite sheet 4x4 (512x512 px total) o iconos individuales
2. Si es sprite sheet:
   - Usar herramienta de edición de imagen para recortar cada icono
   - Exportar cada uno como webp individual
3. Colocar todos los archivos .webp en esta carpeta (assets/icons/)
4. Los nombres de archivo deben coincidir exactamente con los listados arriba

## Paleta de Colores Sugerida

- **Esenciales:** Tonos azules, verdes, rojos (prioridad visual)
- **Recomendables:** Tonos naranjas, amarillos (moderación)
- **No esenciales:** Tonos grises, morados (menor prioridad)

## Prompt para IA Generativa

```
A clean icon sprite sheet on white background with 16 simple flat design icons in 4x4 grid layout. Each icon 128x128 pixels. Icons needed: water bottle, canned food, flashlight, batteries, first aid kit with red cross, emergency whistle, portable radio, documents/ID cards, medical face mask, blanket/sleeping bag, money bills, house keys, laptop computer, hair dryer, thick book, teddy bear plushie. Minimalist flat design style, colorful, clear spacing between icons, professional emergency kit theme.
```
